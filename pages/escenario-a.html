Red de distribución:
La Paz bajo bloqueo

Contexto real: Tres plantas (Senkata, Cochabamba, Oruro) abastecen tres macro-zonas (El Alto, Centro, Sur). El bloqueo de la carretera Oruro–La Paz ha reducido la capacidad de transporte en un 80%. Se modela la distribución óptima de carburante respondiendo: ¿cuánto enviar a cada zona?, ¿qué zona queda más afectada?, ¿el sistema es estable?

Algoritmo LU · Doolittle 
// Descomposición A = L·U
para k = 0 hasta n-1:
  U[k][j] = A[k][j] - Σ L[k][p]*U[p][j]
  L[i][k] = (A[i][k] - Σ ...) / U[k][k]
// Sustitución progresiva:  L·y = b
// Sustitución regresiva:   U·x = y
📌 Método directo · Solución exacta en O(n³) · Ideal para sistemas pequeños y bien condicionados.

Algoritmo Jacobi 
para iter = 1..maxIter:
  para i = 0..n-1:
    x_new[i] = (b[i] - Σ_{j≠i} A[i][j]*x[j])
               / A[i][i]  ← usa x_old
  error = ||x_new - x|| / ||x_new||
  x = x_new
  si error < tol: detener
⚠️ Actualización simultánea · Requiere diagonal dominante · Convergencia lenta pero paralelizable.

Gauss-Seidel 
para iter = 1..maxIter:
  para i = 0..n-1:
    suma = Σ_{j<i} A[i][j]*x[j]  ← x nuevo
         + Σ_{j>i} A[i][j]*x[j]  ← x viejo
    x[i] = (b[i] - suma) / A[i][i]
  error = ||Δx|| / ||x||
✓ Convergencia ~2× más rápida que Jacobi · Usa los valores actualizados inmediatamente.

SOR · Successive Over-Relaxation 
x_gs = (b[i] - Σ A[i][j]*x[j]) / A[i][i]
x[i] = (1−ω)·x[i] + ω·x_gs

// ω = 1    → Gauss-Seidel clásico
// 1<ω<2  → sobrerrelajación (acelera)
// ω < 1  → subrelajación (estabiliza)
⚙️ ω óptimo ≈ 1.1–1.5 para la mayoría de sistemas · Prueba distintos valores y compara iteraciones.

Gradiente Conjugado 
r = b − A·x;  p = r;  rr = r·r
para iter:
  Ap = A·p
  α  = rr / (p·Ap)
  x  = x + α·p
  r  = r − α·Ap
  β  = rr_new / rr
  p  = r + β·p
⚠️ Requiere A simétrica definida positiva · El algoritmo la simetriza automáticamente si no lo es.
