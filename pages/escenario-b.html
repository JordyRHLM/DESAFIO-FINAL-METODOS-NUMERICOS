Vaciado de reservas:
¿Cuándo se acaba el carburante?

Contexto real: Las reservas de carburante en La Paz disminuyen cuando el reabastecimiento se corta por bloqueos y la demanda sube por pánico. Se modela la ecuación R'(t) = entrada − consumo₀·ekt y se comparan tres métodos numéricos para predecir el día en que la reserva llega al nivel crítico.

Euler · O(h) Heun · O(h²) RK4 · O(h⁴)

Modelo diferencial

Modelo · R'(t) = entrada − consumo₀·eᵏᵗ 
// R(t)      = reserva disponible en miles de litros
// entrada   = reabastecimiento diario constante (miles L/día)
// consumo₀  = consumo base inicial (miles L/día)
// k         = tasa de crecimiento del consumo (pánico/demanda)
// Si entrada < consumo₀ · e^(k·t) → la reserva disminuye

// Solución analítica exacta:
R(t) = R₀ + entrada·t − (consumo₀/k)·(eᵏᵗ − 1)
Euler
R_{n+1} = R_n + h·f(t_n, R_n)
Error global O(h) · 1 evaluación/paso
Heun
k1 = f(t,R) · k2 = f(t+h, R+h·k1)
R_{n+1} = R_n + h/2·(k1+k2)
Error global O(h²) · 2 evaluaciones/paso
RK4
Combina k1,k2,k3,k4 con pesos 1,2,2,1
R_{n+1} = R_n + h/6·(k1+2k2+2k3+k4)
Error global O(h⁴) · 4 evaluaciones/paso

Parámetros de simulación

Reabastecimiento constante
0 = consumo constante · 0.07 = pánico
Menor h = más preciso pero más lento
Por debajo = alerta de desabastecimiento

Consumo diario vs. Reabastecimiento

Cuando la curva de consumo supera la línea de entrada, la reserva comienza a disminuir. El cruce de las dos líneas es el punto de inflexión crítico.
