Umbrales críticos:
¿Cuándo colapsa el sistema?

Contexto real: El día exacto en que el consumo supera al reabastecimiento, el precio rebasa el presupuesto familiar, o el descontento social llega al punto de masificación son raíces de ecuaciones no lineales. Se comparan tres métodos con distintas velocidades de convergencia.

Metodos utilizados:

Bisección · O(log₂n) Newton-Raphson · Cuadrático Secante · Superlineal

Algoritmos de búsqueda de raíces

Bisección 
// Requiere: f(a)·f(b) < 0
mientras (b−a)/2 > tol:
  c = (a + b) / 2
  si f(a)·f(c) < 0:
    b = c
  sino:
    a = c
// Garantizado · lineal · lento
Newton-Raphson 
// Requiere: x₀ cerca de raíz
mientras |Δx| > tol:
  x = x − f(x)/f'(x)

// f'(x) por diferencias centrales:
f'(x) ≈ (f(x+h)−f(x−h))/(2h)
// Cuadrático · muy rápido
Secante 
// Requiere: x₀, x₁ iniciales
mientras |Δx| > tol:
  x₂ = x₁ − f(x₁)·(x₁−x₀)
            / (f(x₁)−f(x₀))
  x₀ = x₁; x₁ = x₂
// Superlineal (~1.618) · sin f'

Configuración

Ejemplos:   x*x - 4  ·  Math.exp(x) - 3*x  ·  50 - 80*Math.exp(0.05*x)

Velocidad de convergencia — escala logarítmica

Una pendiente más pronunciada = convergencia más rápida. Newton-Raphson (cuadrático) desciende mucho más rápido que Bisección (lineal). Secante queda entre los dos.
