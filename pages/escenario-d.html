Costo acumulado:
¿Cuánto gasta una familia al mes?

Contexto real: En La Paz, el gasto diario en la canasta básica no es constante — sube con los bloqueos, el pánico de compra y la inflación. La integral ∫c(t)dt sobre el mes calcula el costo total acumulado y permite estimar cuándo el gasto supera el salario familiar.

Trapecio · O(h²) Simpson 1/3 · O(h⁴) Simpson 3/8 · O(h⁴)

Algoritmos de integración numérica

Regla del Trapecio · O(h²) 
// ∫f dx ≈ h/2·[f(x₀) + 2f(x₁) +
//         ... + 2f(xₙ₋₁) + f(xₙ)]
h = (b−a)/n
suma = f(a) + f(b)
para i=1..n-1:
  suma += 2·f(a + i·h)
resultado = (h/2)·suma

// Approx. lineal en cada segmento
// Error global: O(h²)
Simpson 1/3 · O(h⁴) 
// n debe ser PAR
// ∫f dx ≈ h/3·[f(x₀) + 4f(x₁) +
//    2f(x₂) + 4f(x₃) + ... + f(xₙ)]
para i=1..n-1:
  coef = (i % 2 == 0) ? 2 : 4
  suma += coef·f(xᵢ)
resultado = (h/3)·suma

// Approx. parabólica · Error: O(h⁴)
Simpson 3/8 · O(h⁴) 
// n debe ser MÚLTIPLO DE 3
// Coeficientes: 1,3,3,2,3,3,2,...,1
para i=1..n-1:
  coef = (i%3 == 0) ? 2 : 3
  suma += coef·f(xᵢ)
resultado = (3h/8)·suma

// Approx. cúbica · Error: O(h⁴)
// Útil cuando n es múltiplo de 3
